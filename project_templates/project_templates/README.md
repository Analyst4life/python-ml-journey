# Project Templates

This folder contains reusable Jupyter/Colab notebook templates for machine learning projects.

Templates included:

- `ml_template.ipynb` — General ML workflow  
- `eda_template.ipynb` — Exploratory Data Analysis  
- `classification_template.ipynb` — Classification projects  
- `regression_template.ipynb` — Regression projects

Use these templates to start new ML projects quickly and maintain consistency.
